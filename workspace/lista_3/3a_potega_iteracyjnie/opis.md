# Potęga - Iteracyjnie - Wyjaśnienie

## Opis
Algorytm oblicza a^n iteracyjnie, gdzie a to podstawa, n to wykładnik. Wynik jest iloczynem n kopii liczby a.

## Dane Wejściowe
- **a**: podstawa (liczba rzeczywista)
- **n**: wykładnik (liczba całkowita nieujemna, n ≥ 0)

## Wynik
**a^n**: a do potęgi n

## Definicja
- a^0 = 1
- a^n = a × a × ... × a (n razy)

## Przykład
- 2^5 = 32
- 3^3 = 27
- 10^0 = 1

## Zastosowania
- Matematyka
- Fizyka
- Informatyka (potęgi liczby 2)
- Algorytmy (analiza złożoności)

## Uwagi
- **n = 0**: zawsze zwraca 1
- **a = 0**: a^0 = 1, a^n = 0 dla n > 0
- **Ujemne a**: znaki się zmienia w zależności od parzystości n
